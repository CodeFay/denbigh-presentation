  1. Project Kickoff

    Goals

    Partner with client to deliver an intuitive web application for consuming SEC 13F information to simplify and automate client value investing strategy.

  2. The MVP

    1Q19 Milestone

    Present an MVP that meets the initial requirements of the client.


    1. Create database architecture to store 8 quarters of historical data for 5 chosen hedge funds selected by client

    2. Query finance data APIs and load database

    3. Calculate QoQ changes in holdings of selected funds

    4. Present table with stocks purchased by 2+ funds, % ownership of stock within each fund

    Technical details: Setup architecture on server, Populate database with 13F filings using existing API (WhaleWisdom or Other) or custom webscraper for SEC site, Node, Express, React/Gatsby Build Tools

  3. Dynamic Data & Intuitive User Experience

    2Q19 Milestone

    Increase the sophistication of the data analysis while improving the user experience.


    1. Create interface allowing client to modify hedge funds based on CID

    2. Define and design user interface

    3. Continue to refine database and server setup as needed

    Technical details: Ensure dynamic query of holdings based on changes to hedge funds selected, Refine database and server setup, Refine UI with Bootstrap or Material UI

  4. Preparing for the Partners

    3Q19 Milestone

    Present a page for partners to view client's performance


    1. Ensure application is mobile-friendly

    2. Define and create data visualizations

    3. Add additional ratios requested by client

    Technical details: Establish routes for different user accounts, Implement PWA that persists client-side app data in case of network outage, Design and implement algorithm for calculating fund performance, Add additional stock ratios
