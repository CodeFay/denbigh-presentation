  1. Project Kickoff

    Goals

    Partner with client to deliver an intuitive web application for consuming SEC 13F information for two main use cases:


    1. Simplify and automate client value investing strategy

    2. Present information to partners regarding company purchases

  2. The MVP

    1Q19 Milestone

    Present an MVP that meets the initial requirements of the client.


    1. Load database with 8 quarters of historical data for 5 chosen hedge funds selected by client

    2. Calculate QoQ changes in holdings of selected funds

    3. Present table with stocks purchased by 2+ funds, % ownership of stock within each fund

    Technical details: Setup architecture on Heroku, Populate MySQL database with 13F filings using existing API (WhaleWisdom or Other) or custom webscraper for SEC site, Node, Express, React/Gatsby Build Tools

  3. Dynamic Data & Intuitive User Experience

    2Q19 Milestone

    Increase the sophistication of the data analysis while improving the user experience.


    1. Create user identity and page permissions for admin

    2. Create interface allowing client to modify hedge funds based on CID

    3. Continue to refine database and server setup as needed

    4. Define and design user interface

    Technical details: Create user accounts with Firebase, Ensure dynamic query of holdings based on changes to hedge funds selected, Refine MySQL and Heroku server setup, Refine UI with Bootstrap or Material UI

  4. Preparing for the Partners

    3Q19 Milestone

    Present a page for partners to view client's performance


    1. Populate partner facing page with client performance

    2. Ensure application is mobile-friendly

    3. Refine data visualizations

    4. Add additional ratios requested by client

    Technical details: Establish routes for different user accounts, Implement PWA that persists client-side app data in case of network outage, Design and implement algorithm for calculating fund performance, Add additional stock ratios
